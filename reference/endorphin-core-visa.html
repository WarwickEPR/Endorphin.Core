Endorphin.Core

Visa

Namespace: Endorphin.Core

Functions for creating connections and talking to VISA instruments in raw IO mode. Formatted IO using the VISA formatting options is not supported.

Nested types and modules

TypeDescription

A VISA instrument, capable of reading, writing and querzRying strings and byte arrays.

Signature: visaId:string -> timeout:int<MeasureProduct<ms,MeasureOne>> -> writeDelay:int<MeasureProduct<ms,MeasureOne>> option -> Instrument

Open a GPIB VISA instrument with the given VISA identifier address and a timeout in milliseconds to wait for each raw command. The optional write delay is used to delay further communication with the hardware after each write command for a specified period.
